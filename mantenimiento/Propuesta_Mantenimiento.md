
# Propuesta de Mantenimiento
Plan de mantenimiento para el sistema:

## 1. Mantenimiento correctivo 
solución de errores reportado:
- Solucionar fallos al almacenar los datos.
- Arreglar errores en la modificacion da los datos a tabular.
## 2. Mantenimiento evolutivo 
incorporación de nuevas funcionalidades (integraciones, exportes):
- Vinculacion al estado de cuenta del usuario para actualizar los gatos de manera inmediata.
- Actualizaciones constantes para que siempre sea compatible para las nuevas versiones de sistemas operativos o navegadores web.
## 3. Mantenimiento preventivo: 
actualizaciones de dependencias y revisión de seguridad:
- Arreglo de errores constantes para evitar fallos totales del sistema.
- Implementacion de medidas de seguridad en el sistema para evitar vulnerabilidades.


## Frecuencia

- Revisión de dependencias: mensual.
- Backups: semanal.
- Auditoría de seguridad: cada 6 meses.
